═══════════════════════════════════════
MISSION CONTEXT
═══════════════════════════════════════
SensoryScope exists for neurodivergent users — people with autism, ADHD, sensory processing disorder, anxiety, or any condition that makes overwhelming environments painful. The map is the heart of the app. Every pixel should communicate:
  "We understand. This map tells you before you go whether a place will be okay for you."

The three sensory axes are:
  - SOUND (noise level): scored 1-10. 1 = near-silent, 10 = deafeningly loud.
  - LIGHT (lighting intensity): scored 1-10. 1 = dim/soft, 10 = harsh bright fluorescents.
  - CROWD (crowd density): scored 1-10. 1 = empty, 10 = packed wall-to-wall.

Lower scores = calmer = more sensory-friendly. This is THE design principle.

Score thresholds from scoreColor() in theme.ts:
  ≤ 3 = calm = Colors.calm   (#3ab98f light / #10B981 dark) — safe, peaceful, go here
  ≤ 6 = moderate = Colors.moderate (#ce9b43 / #F59E0B) — manageable, prepare yourself
  > 6 = intense = Colors.intense (#d74c64 / #F43F5E) — overwhelming, approach with caution

2. scoreColor() USES LightColors CONSTANTS (NOT DYNAMIC)
   In constants/theme.ts, scoreColor() and scoreGlow() reference Colors.calm/moderate/intense
   which is always LightColors. In dark mode the pins show the wrong colors (light theme greens).

   FIX: Do NOT change theme.ts. Instead, in map.tsx, compute score colors dynamically:

   Create a hook or inline helper that takes (score, C) where C = useColors():
     function getSensoryColor(score: number, C: ReturnType<typeof useColors>): string {
       if (score <= 3) return C.calm;
       if (score <= 6) return C.moderate;
       return C.intense;
     }
     function getSensoryGlow(score: number, C: ReturnType<typeof useColors>): string {
       if (score <= 3) return C.calmGlow;
       if (score <= 6) return C.moderateGlow;
       return C.intenseGlow;
     }

   MapPin cannot use hooks (it's inside Marker). Instead, pass the computed color as a prop:
     <MapPin score={score} color={computedColor} glowColor={computedGlowColor} isSelected={isSelected} />

   In the parent, compute it before passing:
     const color = score !== null ? getSensoryColor(score, C) : C.textDim;
     const glowColor = score !== null ? getSensoryGlow(score, C) : C.surface;
